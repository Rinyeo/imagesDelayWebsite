使用方法

1. 在html中引入 jQeury 和 ImagesDelay.js：

<script src="jquery-1.11.1.min.js"></script>
<script src="jquery.imagesDelay.js"></script>

2. 选中需要延迟加载的 img ，调用方法 ImagesDelay 方法：

$(function() {
  $('.main img').imagesDelay();
});

3. HTML中class="mian"下的子节点img会被延迟加载，如果想选取部分img，请使用筛选或者给img添加class。同时延迟加载的img应该符合以下规则：
页面初始化时，data-src 属性为延迟加载的最终图片路径。
页面初始化时，src 属性为延迟加载默认的图片路径。

<div><img data-src="delay.jpg" src="defualt.jpg" width="100%"></div>

4. imagesDelay()可以传入一个参数，可以控制游览器窗口底部到img上方距离多少时延时加载，默认为100px。

$(function() {
  $('.main').imagesDelay(-200); //可以传入负值
});
